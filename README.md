# 🧠 General Knowledge Quiz

A beginner-friendly Python project that asks users multiple general knowledge questions, checks their answers, and displays the final score.

## 📌 Features

- User-friendly quiz interface
- Multiple General Knowledge questions
- Input validation using `strip()` and `lower()`
- Option to exit the quiz anytime
- Final score display
- Beginner-friendly Python logic

## 🛠 Technologies Used

- Python 3

## 📚 Concepts Practiced

- User Input
- Conditional Statements (if/else)
- Variables
- String Methods (`strip()` & `lower()`)
- Score Management
- Input Validation

## ▶️ How to Run

1. Clone this repository.
2. Open the project in your preferred Python IDE.
3. Run the Python file.
4. Answer the quiz questions or type **exit** to quit.
5. View your final score.
